So far this list only covers the TI-83+ and TI-84+. If anyone is willing to write up some more pages for other models, feel free to PM me on Cemetech and I will proofread and add it.

Great Websites

  • Cemetech - Great community focused on graphing calculators (and trains, apparently?)
  • TI-Planet - Most news gets posted here first. Also it's French.
  • TICalc - Most people only use this for file archives now, but occasionally there are news posts there.
  • TI|BD - A mostly-inactive but still great resource for TI-BASIC programmers.
  • CodeWalrus - Like Cemetech but less active.
  • Omnimaga - Like Cemetech and CodeWalrus, but still less active.

Must-have TI-83+/TI-84+ Programs

  • (Download) Axe - A compiled language similar to TI-BASIC.
  • (Download) CalcSys - Swiss army knife of calculator tools.
  • (Download) DoorsCS - Catch-all shell with built-in GUI libs.
  • (Download) zStart - Customize your calulator and recover after RAM clears.
